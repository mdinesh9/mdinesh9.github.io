Steps:

  • Create an Azure subscription

Setup

Tools: - azure sdk - azure function core tools - pip install azure-functions

Commands: - Azure function tools - func - func new --template "EventGridTrigger"

Azure Functions, is a serverless computing service which can run code without managing infrastructure fro our end and it can be triggered by different azure events, such as events detected in Blob storage.

Azure Blob Trigger, can trigger code(azure function), on event like Create, delete, modification a blob

Azure Event Grid, is a fully managed event routing service which utilizes the publish/subscribe pattern, and in our case, we can detect when a blob is detected or modified on azure blob storage, and send a message about this event to azure blob trigger. Here, Azure Event Grid, subscribes to Azure Blob Trigger, and publishes it to Azure Function, which then will run the indexing pipeline.
